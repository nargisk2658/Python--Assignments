# Flask Registration System

This project is a simple user registration system built using Flask.

## Features
- User Registration Form
- Form Validation using Flask-WTF
- Confirm Password validation
- Data stored in SQLite database
- Flash message for duplicate email

## Installation

Install required packages:

pip install -r requirements.txt

## Run the Application

python app.py

Then open:

http://127.0.0.1:5000/

The database file (users.db) will be created automatically.
